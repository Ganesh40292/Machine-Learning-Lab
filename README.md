# 🧠 Machine Learning Lab

This repository contains implementations of fundamental Machine Learning algorithms and data analysis techniques as part of an academic Machine Learning Lab. It covers core concepts such as exploratory data analysis, supervised and unsupervised learning, dimensionality reduction, and probabilistic models.

---

## 📌 Objectives

- Understand machine learning concepts through practical implementation  
- Perform exploratory data analysis and visualization  
- Implement classical ML algorithms  
- Analyze and interpret model performance  

---

## 🛠️ Technologies Used

- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn

## 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/your-username/Machine-Learning-Lab.git
```
2. Navigate to the folder:
```bash
cd Machine-Learning-Lab
```
3. Install dependencies:
```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

4. Run any experiment:
   ```bash
   python filename.py
   ``
