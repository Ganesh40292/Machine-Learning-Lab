🧠 Machine Learning Lab

This repository contains implementations of fundamental machine learning algorithms and data analysis techniques as part of a structured Machine Learning Lab curriculum. The experiments cover a wide range of topics including exploratory data analysis, supervised and unsupervised learning, dimensionality reduction, and probabilistic models.

📌 Objectives
Understand core machine learning concepts through hands-on implementation
Perform exploratory data analysis and visualization
Implement classical machine learning algorithms from scratch and using libraries
Analyze and interpret model performance on real-world datasets

🛠️ Technologies Used
Python 🐍
NumPy
Pandas
Matplotlib
Seaborn
Scikit-learn

📂 Experiments Overview
🔹 1. Exploratory Data Analysis (EDA)
Created histograms and box plots for numerical features
Identified distribution patterns and outliers
Dataset: California Housing
🔹 2. Feature Relationships & Visualization
Computed correlation matrix
Visualized using heatmaps
Generated pair plots for feature relationships
Dataset: California Housing
🔹 3. Dimensionality Reduction
Applied Principal Component Analysis (PCA)
Reduced dataset from 4D to 2D
Dataset: Iris
🔹 4. Find-S Algorithm
Implemented concept learning algorithm
Derived hypothesis consistent with training examples
🔹 5. k-Nearest Neighbors (k-NN)
Classified randomly generated data points
Tested model performance for multiple values of k
🔹 6. Locally Weighted Regression
Implemented non-parametric regression
Visualized regression fit for selected dataset
🔹 7. Regression Models
Linear Regression on Boston Housing dataset
Polynomial Regression on Auto MPG dataset
🔹 8. Decision Tree Classifier
Built classification model
Applied on Breast Cancer dataset
🔹 9. Naive Bayes Classifier
Trained model on Olivetti Face dataset
Evaluated classification accuracy
🔹 10. K-Means Clustering
Implemented unsupervised clustering
Visualized clusters on dataset

🚀 How to Run

Clone the repository:

git clone https://github.com/your-username/Machine-Learning-Lab.git

Navigate to the project directory:

cd Machine-Learning-Lab

Install required libraries:

pip install numpy pandas matplotlib seaborn scikit-learn

Run any experiment:

python experiment_name.py
